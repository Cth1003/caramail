Votre navigateur n'est pas configuré pour supporter JAVA. Toutefois rien n'est perdu ! Si vous disposez d'un navigateur récent (version 3, 4 ou supérieure), veuillez vérifier que JAVA est bien interprété. Pour cela allez dans les préférences de votre navigateur puis :
  • cochez les cases relatives à JAVA,
  • quittez,
  • redémarrez votre navigateur.
